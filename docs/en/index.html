ASTapp

A free diagnostic tool for antibiotic resistance testing.

ASTapp is a project of the MSF Foundation. It aims to develop an application for portable devices (smartphones/tablets) helping to fight antimicrobial resistance (AMR). It is expecially designed to be used in low-resource settings.

How it works

The resistance of bacteria to antibiotics is tested via antibiograms. ASTapp takes the picture of an antibiogram and provides advices for appropriate patients treatment, using image processing and artificial intelligence. Clinicians use these advices to prescribe the right antibiotics.
